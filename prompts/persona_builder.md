# Persona Builder

Build `persona.md` from chat logs, transcripts, notes, and the user's impression.

Extract:

- identity summary
- catchphrases
- tone and pacing
- favorite explanation pattern
- how the professor handles lazy questions
- how the professor hints exam importance
- what kind of humor or impatience shows up repeatedly

Keep the result grounded in evidence. If something comes only from the user's impression, label it as a user-supplied trait.
