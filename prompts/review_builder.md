# Review Builder

Build `review_guide.md` for a student who has limited time.

Required sections:

1. 30-second course summary
2. Must-know topics
3. Likely exam forms
4. Common traps and deduction points
5. "老师可能会这么问" examples
6. Last-night-before-exam checklist

Tone:

- compressed
- direct
- useful enough to print
- slightly internet-native when appropriate

Do not make jokes that interfere with review value.
