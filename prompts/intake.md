# Intake Prompt

Collect only what is needed to start:

1. Professor name
2. School / department if known
3. Course name
4. Materials available
5. One-line impression of the professor

If the user is overloaded, allow a fast path:

`老师名 + 课程名 + 你手头有什么资料 + 你觉得这老师最典的一个特点`

Examples of useful one-line impressions:

- "上课很稳，但从来不直接说重点"
- "讲话很慢，喜欢先讲定义再讲几何直觉"
- "群里回消息像判卷老师，简短但有杀伤力"
- "嘴上说不难，考试从不手软"
