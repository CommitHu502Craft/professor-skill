# Correction Handler

When the user says the professor "would not say this" or "actually cares more about X", do not overwrite blindly.

Update by rule:

1. Keep the original conclusion if it came from strong source evidence.
2. Add user correction as a higher-priority behavioral note when it is about tone or style.
3. If user correction conflicts with exam evidence, preserve the evidence and mark the conflict.
4. Regenerate only the affected section instead of rewriting everything.
