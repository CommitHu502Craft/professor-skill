# Materials Schema

Use this reference when organizing raw teaching materials before writing the final professor skill.

## Recommended directory layout

```text
materials/
├── slides/
├── syllabus/
├── exams/
├── assignments/
├── transcripts/
├── chats/
├── notices/
└── misc/
```

## Source priority

Higher-priority sources should dominate exam and review conclusions:

1. `exams/`
2. `assignments/`
3. `transcripts/`
4. `slides/`
5. `syllabus/`
6. `chats/`
7. `notices/`
8. `misc/`

## Material labeling suggestions

Prefer filenames that reveal time and type:

- `2025-final-exam.pdf`
- `week07-eigenvalues-slides.pdf`
- `lecture-09-transcript.md`
- `office-hour-qa-2025-12-18.txt`
- `group-chat-important-points.txt`

## Extraction hints

- Slides reveal structure, not always importance.
- Exams and assignments reveal actual pressure points.
- Transcripts reveal explanation order and catchphrases.
- Chat logs reveal what students repeatedly fail to understand.
- Notices reveal hidden constraints, schedule changes, and "this is important" moments.
